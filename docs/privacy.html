プライバシーポリシー

最終更新日: 2026年1月10日

重要: この拡張機能は一切のユーザーデータを収集・保存・送信しません。全ての処理はあなたのブラウザ内でローカルに完結します。

概要

「NotoSansへ置換するやつ(改修型)」（以下「本拡張機能」）は、ウェブページ上の読みづらい日本語フォントを自動的にNoto Sansフォントに置換するブラウザ拡張機能です。

本プライバシーポリシーは、本拡張機能がユーザーの情報をどのように扱うかを説明するものです。

データ収集について

収集しないデータ

本拡張機能は、以下を含むいかなるユーザーデータも収集しません：

  • 個人情報（名前、メールアドレス、住所など）
  • ブラウジング履歴
  • 訪問したウェブサイトのURL
  • 入力されたテキストやフォームデータ
  • Cookie情報
  • IPアドレス
  • デバイス情報
  • 位置情報
  • その他いかなる種類のユーザーデータ

拡張機能の動作

処理内容

本拡張機能は、以下の処理をローカルで実行します：

  • ウェブページのフォント設定を検出
  • 特定の日本語フォント（MS Gothic、Meiryo、Yu Gothicなど）を検出した場合、Noto Sansフォントに置換
  • 動的に追加されるiframe内のフォントも同様に置換

重要な点

  • 全ての処理はブラウザ内で完結：外部サーバーへの通信は一切行いません
  • フォントファイルは拡張機能内に同梱：外部からのフォント読み込みは行いません
  • データの保存なし：ブラウザのローカルストレージやサーバーへのデータ保存は行いません

権限について

ホスト権限（<all_urls>）

本拡張機能は、全てのウェブサイトへのアクセス権限を要求しますが、これは以下の理由によるものです：

  • ユーザーが訪問するあらゆるウェブページでフォント置換を行うため
  • 各ページのフォント設定を検出し、CSSを動的に注入するため

この権限は、フォント置換処理のためだけに使用され、データの収集や送信には一切使用されません。

第三者への情報提供

本拡張機能は、いかなるデータも収集しないため、第三者への情報提供は一切ありません。

  • 広告ネットワークへの情報提供なし
  • 分析ツールの使用なし
  • 外部サービスとの連携なし

Privacy Policy

Last Updated: January 10, 2026

Important: This extension does not collect, store, or transmit any user data. All processing is done locally within your browser.

Overview

"NotoSansへ置換するやつ(改修型)" (Replace with Noto Sans - Improved Version, hereinafter "this extension") is a browser extension that automatically replaces hard-to-read Japanese fonts on web pages with Noto Sans font.

This privacy policy explains how this extension handles user information.

Data Collection

Data We Do NOT Collect

This extension does not collect any user data, including but not limited to:

  • Personal information (name, email address, physical address, etc.)
  • Browsing history
  • URLs of visited websites
  • Entered text or form data
  • Cookie information
  • IP addresses
  • Device information
  • Location data
  • Any other kind of user data

How the Extension Works

Processing

This extension performs the following processes locally:

  • Detects font settings on web pages
  • Replaces specific Japanese fonts (MS Gothic, Meiryo, Yu Gothic, etc.) with Noto Sans font when detected
  • Also replaces fonts within dynamically added iframes

Key Points

  • All processing is done within the browser: No communication with external servers
  • Font files are bundled with the extension: No external font loading
  • No data storage: No data is saved to browser local storage or servers

Permissions

Host Permission (<all_urls>)

This extension requests access to all websites for the following reasons:

  • To perform font replacement on any web page the user visits
  • To detect font settings on each page and dynamically inject CSS

This permission is used solely for font replacement and is never used for data collection or transmission.

Third-Party Data Sharing

Since this extension does not collect any data, there is no sharing of information with third parties.

  • No data sharing with advertising networks
  • No use of analytics tools
  • No integration with external services

Open Source

The source code of this extension is publicly available on GitHub:

https://github.com/1llum1n4t1s/replace-font

Anyone can review and verify the code, ensuring transparency regarding privacy and security.

Contact

Policy Changes

This privacy policy may be updated as necessary. Significant changes will be announced through the GitHub repository and extension update notes.
