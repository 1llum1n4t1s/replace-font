# Chrome Web Store スクリーンショット

このディレクトリには、Chrome Web Storeに掲載するためのスクリーンショット用HTMLファイルが含まれています。

## 📸 スクリーンショット一覧

1. **01-popup-ui.html** - 拡張機能のポップアップUI表示
2. **02-before-after.html** - フォント置換のBefore/After比較
3. **03-hero-promo.html** - プロモーション用ヒーロー画像

## 🎯 スクリーンショットの作成方法

### 方法1: ブラウザのスクリーンショット機能を使用

#### Chrome / Edge の場合:

1. HTMLファイルをブラウザで開く
2. F12キーを押して開発者ツールを開く
3. `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`) でコマンドパレットを開く
4. "Capture full size screenshot" と入力して実行
5. 自動的にPNGファイルがダウンロードされます

#### Firefox の場合:

1. HTMLファイルをブラウザで開く
2. F12キーを押して開発者ツールを開く
3. `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`) でコマンドパレットを開く
4. "screenshot" と入力して "Take a full page screenshot" を選択
5. 自動的にPNGファイルがダウンロードされます

### 方法2: ブラウザ拡張機能を使用

以下の拡張機能が便利です:

- **Awesome Screenshot** - https://www.awesomescreenshot.com/
- **FireShot** - https://getfireshot.com/

### 方法3: オンラインツールを使用

HTMLファイルをアップロードしてスクリーンショットを生成:

- **Screely** - https://www.screely.com/
- **Browserframe** - https://browserframe.com/

## 📐 推奨サイズ

Chrome Web Storeの推奨仕様:

- **解像度**: 1280x800 ピクセル（既に設定済み）
- **形式**: PNG または JPEG
- **最大ファイルサイズ**: 各5MB以内
- **必要枚数**: 最低1枚、最大5枚まで

## 🎨 カスタマイズ

各HTMLファイルは独立しているため、必要に応じて以下をカスタマイズできます:

- 色やグラデーション
- テキスト内容
- アイコンやバッジ
- レイアウト

## 📝 使用方法

1. 各HTMLファイルをブラウザで開く
2. 上記の方法でスクリーンショットを撮影
3. 生成されたPNGファイルを保存
4. Chrome Web Store Developer Dashboardにアップロード

## 💡 Tips

- **高品質な画像**: ブラウザのズームレベルを100%に設定してください
- **一貫性**: 全てのスクリーンショットで同じブラウザを使用することを推奨
- **順序**: Chrome Web Storeでは表示順序を設定できるため、番号順にアップロードしてください
